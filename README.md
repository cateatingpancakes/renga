# renga
This repository aims to implement a [Riichi mahjong](https://en.wikipedia.org/wiki/Japanese_mahjong) engine in Java. While this project will serve as my term project for the Java object-oriented programming course at the University of Bucharest, I do intend to at least somewhat maintain it and build as extensive and extensible a framework as possible. 

Minimally, I aim to implement [sanma](https://riichi.wiki/Sanma) and yonma up to common ruleset variations and [lists of yaku](https://riichi.wiki/Local_yaku), while leaving the possibility of implementing other game modes like Sichuan bloody (try it, it's fun) or all those weird Jantama event ones open for anyone.

# how to use
This repository will be packaged with a server running the game engine and a minimal client that connects to it. I'll add a tutorial and a list of hardware/software requirements for those later.
